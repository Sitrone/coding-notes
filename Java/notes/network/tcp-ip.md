### TCP 

> 传输控制协议，是一种面向连接的、可靠的、基于字节流的传输层通信协议。TCP 负责在不可靠的传输信道之上提供可靠的抽象层，向应用层隐藏了大多数网络 通信的复杂细节，比如丢包重发、按序发送、拥塞控制及避免、数据完整，等等。
